About Me

Hello, I'm Joie Harvey, a full-stack web developer with a passion for building secure and functional web applications. I specialize in both front-end and back-end development with the use of Python frameworks like Django and Flask for back-end logic, and HTML, CSS, Javascript, and React.js for creating intuitive user interfaces. I also perform penetration tests using tools such as, Kali Linux and burp suite, to ensure that each developed website is secure and resistant to vulnerabilities. I aspire to assist businesses in creating or updating their websites to ensure an optimal user experience through security, functionality, and modern design. Feel free to reach out with any questions or for assistance on your next project. I specialize in building full-stack eCommerce sites, blogs, social platforms, and custom solutions tailored to meet any personal or business requirements.

Flask Firewall

A firewall that was built with the purpose of protecting Flask web applications from common web vulnerabilities and attacks, such as DDoS, brute force, payload injections, and CSRF attacks through enforcing rate limits and login rate limits within login POST requests, input validation and output encoding, CSRF token generation and validation, and IP whitelisting and blacklisting for handling any offenders or granting specific access.

Technologies Used: Python, Burp Suite(penetration testing), Kali Linux(penetration testing)

Machine Learning Market Predictor App

An autoencoder model that was built and trained with Tensorflow to detect patterns within historical price data gathered from YFinances API. After training, a real-time price is fetched from yfinance's market API, and the model uses its learned patterns to predict the future price based on the current price. The model's logic is integrated into a Flask web application to allow users to customize the training process based on their preferred asset symbol.

Technologies Used: Tensorflow, HTML, CSS, Flask, Python, yfinance
