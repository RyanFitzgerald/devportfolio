Bomb Defusal Game

This is a 2 or more player cooperative game. One player is the bomb defuser and the other players take on the role of the bomb defusal specialists.

The bomb defuser must relay what he/she is seeing on the bomb and the specialists will have to piece together a solution based off of their bomb defusal manuals.

This game is based off of “Keep Talking and Nobody Explodes”, a popular co-op game.

Made for my Game Design class in November 2017 using Arduino.

Matrix Filter

This is a Touch Designer project that takes in a video input and converts it to a set of height values for a Grid. Based off the luminance value of a set of pixels it will raise or lower a pixel on the grid.

Originally uses Kinect for video and motion controls to provide user input. Shown with a 2016 Macbook Pro camera in video.

Made for my Accio class in February 2018 using Touch Designer.

Audio Reactive Visualization with TouchOSC Remote

This is a audio reactive visualization that creates a set of particles based off a shape input and an audio input.

It is entirely controlled by TouchOSC interface which allows the viewer to change the base shape, color, shape details, audio input, and tilt the geometry using the acceleromter of the phone.

Made for my Accio class in March 2018 using Touch Designer.

Useless Toilet Paper Machine

This is a useless machine made to seperate 2 ply toilet paper into a pair of 1 ply rolls. Made from plywood, Arduino Uno, and custom 3D printed dowels.

It is programmed with three states cycled by a push button

1. Startup state where it plays a chime and a breathing effect using LEDs
2. Forward state where it spins the motors to seperate the toilet paper
3. Reverse state where the motors are reversed.

Made for my Physical Computing class in March 2018 using Arduino and Rhinceroes.
