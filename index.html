    PCGCTF

    PCGTF stands for Procedural Content Generation for Capture The Flag Competitions. This is an on-going research project to create a program that makes realistic virtual network environments for CTF penetration-testing competitions.
    The game topology and vulnerability path are generated in Python, then the resulting configuration is used to generate a DETER configuration file for use in a DETERLab environment. Currently the 3 core components are in development:

  • Topology Generation

    • A network topology is generated from pre-defined network parameters(number of clients, firewalls, subnets, etc).

  • Vulnerability Injecting

    • Vulnerability types and locations are assigned to clients on the network algorithmically to define a "game path" for the user to follow.

  • Visualization/Representation

    • Human readable and computer-readable representations of the network are generated. The computer representation is in the form of a DETER configuration file for network setup and generation.

    AirMonkey 2

    AirMonkey 2 is a WiFi auditing tool with an Android App interface. Connected via a WiFi hotspot, an Android phone issues commands to a Raspberry Pi to perform various network auditing routines, such as:

  • AP enumeration
  • De-Auth Attack Vector
  • EvilAP Attack Vector

    • Compared to similar tools, AirMonkey 2 offers reduced configuration and setup times without sacrificing performance or customizability.
